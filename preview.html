How to Import into Figma

The fastest way to get CarbonS into Figma is through the Tokens Studio plugin. Here's the complete workflow:

1
Install Tokens Studio — Search "Tokens Studio for Figma" in Figma's Community plugins (1M+ installs, free tier available).
2
Download the tokens file — Click "Download for Figma" above to save tokens-studio.json to your computer.
3
Open the plugin — In your Figma file, run Tokens Studio. Go to the Settings tab (gear icon).
4
Load tokens — Under "Token storage", choose Local document. Then use the import icon and select the downloaded JSON file.
5
Apply to Figma — Check the token sets you want (Primitives, Semantic/Light, Typography, etc.) and click "Create variables" or "Create styles". Done.
Pro tip: For a GitHub-synced workflow, point Tokens Studio to the raw URL: https://raw.githubusercontent.com/borisj74/carbons-design-system/main/figma/tokens-studio.json This keeps your Figma variables in sync with your codebase automatically.
